# Objetivo
Este projeto foi feito como ferramenta para auxiliar na leitura de documentos de identidade brasileira.
O projeto busca homologar outros documentos ex: CNH, CLT, ETC...

Atualmente Testado apenas com a identidade brasileira modelo 2018, porem deve funcionar com as demais.

# Como instalar
1 - Clone o repositorio
<br>
2 - Execute pip install -r requirements.txt (Python 3.12)

# Utilização
1 - Execute o arquivo scan_id.py
<br>
2 - Envie uma imagem para o endpoint /scan
<br>
3 - Receba o JSON com os dados extraídos
